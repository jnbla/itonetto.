# Panduan Instalasi dan Penggunaan IkiNet

Dokumen ini menjelaskan langkah instalasi dan penggunaan aplikasi booking lapangan IkiNet bagi pengguna biasa dan admin.

## 1. Persiapan Sistem

### 1.1. Requirement
- Sistem operasi Windows
- XAMPP terpasang dan berjalan
- Apache dan MySQL aktif
- Browser modern (Chrome/Edge/Firefox)

### 1.2. Struktur Direktori
- `app/controllers` - controller aplikasi
- `app/models` - model dan query database
- `app/views` - tampilan halaman
- `app/helpers` - fungsi helper
- `config/database.php` - konfigurasi koneksi database
- `database/schema.sql` - struktur database
- `uploads/` - tempat penyimpanan file gambar lapangan
- `public/` - root web untuk diakses melalui browser

## 2. Langkah Instalasi

### 2.1. Salin Proyek ke XAMPP
1. Salin folder proyek ke:
   `C:\xampp\htdocs\IkiNet`
2. Pastikan folder `uploads` dapat ditulis oleh Apache.

### 2.2. Jalankan Apache dan MySQL
1. Buka `XAMPP Control Panel`.
2. Klik tombol `Start` pada `Apache`.
3. Klik tombol `Start` pada `MySQL`.

### 2.3. Buat Database
1. Buka `http://localhost/phpmyadmin`.
2. Buat database baru bernama `badminton_booking`.
3. Pilih database `badminton_booking`.
4. Import file `database/schema.sql`.

### 2.4. Periksa Konfigurasi Database
1. Buka file `config/database.php`.
2. Pastikan nilai berikut sesuai:
   - `$host = "localhost";
   - `$user = "root";`
   - `$pass = "";`
   - `$db   = "badminton_booking";`
3. Jika memakai username/password MySQL berbeda, ubah sesuai.

### 2.5. Akses Aplikasi
- Buka URL:
  `http://localhost/IkiNet/public/`

## 3. Penggunaan untuk Admin

### 3.1. Login dan Registrasi Admin
- Jika belum ada akun, daftar akun baru.
- Akun pertama yang dibuat akan otomatis menjadi `admin`.
- Setelah login sebagai admin, Anda akan melihat menu admin.

### 3.2. Dashboard Admin
Pada menu admin, Anda dapat:
- Melihat statistik dan ringkasan data.
- Mengakses `Lapangan`, `Reservasi`, `Analytics`, `Audit Log`, dan `Settings`.

### 3.3. Kelola Lapangan
Admin bisa:
- Menambahkan lapangan baru.
- Mengedit data lapangan.
- Menghapus lapangan.
- Mengunggah foto lapangan.

### 3.4. Kelola Booking
Admin bisa:
- Melihat semua booking user.
- Mengubah status booking menjadi:
  - `Menunggu`
  - `Disetujui`
  - `Dibatalkan`
  - `Selesai`
- Melihat detail reservasi.

### 3.5. Settings Aplikasi
Pada halaman `Settings`, admin dapat mengatur:
- Nama aplikasi
- Nama instansi
- Kontak admin
- Integrasi WhatsApp API

## 4. Penggunaan untuk Pengguna

### 4.1. Registrasi dan Login
- Akses `http://localhost/IkiNet/public/`.
- Daftar akun baru dengan `username`, `email`, dan `password`.
- Login setelah akun dibuat.

### 4.2. Booking Lapangan
Pengguna dapat:
- Masuk ke halaman `Booking Baru`.
- Memilih lapangan, tanggal, jam mulai, dan jam selesai.
- Menambahkan catatan dan nomor WhatsApp.
- Menekan tombol `Ajukan Booking`.
- Sistem otomatis memeriksa jadwal bentrok.

### 4.3. Melihat Booking Saya
- Akses menu `Booking Saya`.
- Melihat ringkasan booking dalam status:
  - `Menunggu`
  - `Disetujui`
  - `Selesai`
  - `Dibatalkan`
- Membatalkan booking sendiri jika diperlukan.

### 4.4. Halaman Profil
Pengguna bisa mengubah:
- Username
- Email
- Password

### 4.5. QR Code Booking
- Setiap booking akan tersedia QR code untuk bukti reservasi.
- QR code dibuat menggunakan layanan API eksternal.

## 5. Catatan Penting

- Pastikan folder `uploads/` dapat ditulis.
- Jika menggunakan port Apache selain default, sesuaikan URL.
- Pastikan service MySQL aktif saat aplikasi digunakan.
- Akun pertama yang membuat registrasi berperan sebagai `admin`.

## 6. Troubleshooting

### 6.1. Jika Tidak Bisa Login
- Pastikan database sudah diimport.
- Pastikan tabel `users` berisi data.
- Jika password salah, coba reset dengan membuat akun baru.

### 6.2. Jika Booking Tidak Bisa Disimpan
- Pastikan waktu mulai dan selesai valid.
- Pastikan interval tidak bentrok pada lapangan yang sama.
- Pastikan data lapangan aktif dan tidak terhapus.

## 7. Kontak Dukungan
- Hubungi admin melalui fitur kontak pada `Settings`.
- Pastikan nomor WhatsApp admin terisi jika ingin menggunakan notifikasi WhatsApp.
